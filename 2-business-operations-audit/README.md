# Project 2 — Business Operations Audit

## Overview
Maps and audits the order fulfilment process for a simulated e-commerce business (RetailFlow Ltd). Identifies bottlenecks, calculates error costs, and produces a structured improvement report.

## What it does
- Analyses 8 process steps from order receipt to courier collection
- Identifies bottleneck steps causing delays and errors
- Calculates monthly and annual financial cost of process errors
- Produces actionable recommendations with quantified impact
- Outputs 3-panel visualisation: time breakdown, error rates, cost distribution

## Skills demonstrated
- Business process mapping and analysis
- Operational efficiency measurement
- Financial impact quantification
- Stakeholder-ready report writing

## How to run
```bash
pip install pandas matplotlib numpy
python main.py
```
