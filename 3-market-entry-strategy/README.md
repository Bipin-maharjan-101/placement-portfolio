# Project 3 — Market Entry Strategy

## Overview
Full market entry strategy for a UK fashion retailer (StyleCo) expanding into Germany. Covers PESTLE, SWOT, competitor analysis, market sizing, revenue projections, and entry mode recommendation — framed as a management consulting deliverable.

## Frameworks used
- **PESTLE** — 6-pillar macro environment analysis with impact and priority ratings
- **SWOT** — Strengths, Weaknesses, Opportunities, Threats
- **Competitor Analysis** — Market share, pricing, sustainability benchmarking
- **Entry Mode Evaluation** — Scored across cost, speed, control, and risk

## How to run
```bash
pip install pandas matplotlib numpy
python main.py
```
