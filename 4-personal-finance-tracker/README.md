# Project 4 — Personal Finance Tracker & Analysis

## Overview
A complete 12-month personal finance tracker with income/expense categorisation, savings rate analysis, and a 4-panel dashboard. Demonstrates core accounting, budgeting, and data presentation skills directly relevant to finance roles.

## Features
- Tracks salary, freelance, and other income streams
- Categorises expenses (essential, discretionary, savings)
- Calculates monthly net cashflow and savings rate
- Identifies high-spend months and budget recommendations
- 4-panel dashboard: income vs expenses, savings rate, stacked breakdown, cumulative position

## How to run
```bash
pip install pandas matplotlib numpy
python main.py
```

## Skills demonstrated
- Personal and management accounting
- Budget vs actual tracking
- Savings rate and cashflow analysis
- Financial data visualisation
