# Project 5 — Financial Ratio Analysis

## Overview
Analyses 3 years of annual report data for a simulated FTSE 250 company (RetailGroup PLC). Calculates 20+ financial ratios across profitability, liquidity, efficiency, and gearing — presented in analyst-style format with commentary and investment recommendation.

## Ratios calculated
- **Profitability:** Gross margin, operating margin, net margin, EBITDA margin, ROE, ROA
- **Liquidity:** Current ratio, quick ratio, cash ratio
- **Efficiency:** Asset turnover, inventory days, receivable days
- **Gearing:** Debt-to-equity, debt-to-assets, interest cover
- **Per share:** EPS, DPS, free cashflow per share

## How to run
```bash
pip install pandas matplotlib numpy
python main.py
```

- Financial statement reading and interpretation
- Ratio calculation and trend analysis
