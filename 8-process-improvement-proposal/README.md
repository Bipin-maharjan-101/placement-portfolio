# Project 8 — Process Improvement Proposal

## Overview
A full business analysis project proposing the digitisation of a university's student enrolment process. Applies professional BA tools including MoSCoW prioritisation, requirements documentation, stakeholder analysis, and process mapping — producing a structured proposal that mirrors real BA deliverables.

## BA frameworks applied
- **Current state process mapping** — 8-step pain point analysis
- **Stakeholder power/interest grid** — identifying and prioritising stakeholders
- **MoSCoW prioritisation** — categorising requirements by importance
- **Requirements document** — functional and non-functional requirements
- **Future state design** — projected improvements with measurable KPIs

## Projected impact
- End-to-end time: 102 hrs → 4.6 hrs (95% reduction)
- Admin effort per student: 4 hrs → <1 hr
- Estimated annual saving: ~£42,000
